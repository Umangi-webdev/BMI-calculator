## 📌 About

The **BMI Calculator** is a responsive web application built using **HTML, CSS, and JavaScript** that helps users calculate their **Body Mass Index (BMI)** based on height and weight inputs.

The application instantly displays the BMI value along with the corresponding health category, helping users better understand their fitness and health status through a clean and intuitive user interface.

## 🧮 BMI Formula

BMI is calculated using:

```text
BMI = Weight (kg) ÷ Height² (m²)
```

Example:

```text
Weight = 70 kg
Height = 1.75 m

BMI = 70 / (1.75 × 1.75)
BMI = 22.86
```
